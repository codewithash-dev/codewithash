# links.cwa

CodeWithAsh's Linktree-style link-in-bio page. A simple, beautiful way to share your links.

## Run locally

Open `index.html` in a browser, or use a simple static server:

```bash
npx serve .
```

Then visit `http://localhost:3000`

## Deploy

This is a static HTML/CSS/JS project. Deploy to:
- **Vercel** – drag & drop the folder
- **Netlify** – same
- **GitHub Pages** – push and enable Pages
- **links.cwa** subdomain – point your domain here after deployment
